## Overview

Architected and developed a fully headless e-commerce platform. It leverages Next.js for server-side rendering, ensuring optimal SEO and performance. Integrated Stripe for secure payment processing and built a custom admin panel for inventory and order management using PostgreSQL and Prisma.
