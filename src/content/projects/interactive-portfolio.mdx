## Overview

This portfolio was built with Next.js 16, Tailwind v4, and framer-motion. It acts as a living sandbox featuring a live code editor, an interactive MDX blog, and dynamic theming. It leverages Server Components for efficient MDX parsing and client components for interactive graphs, achieving a performant, modern frontend experience.
