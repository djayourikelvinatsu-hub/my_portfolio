## Overview

Aura is a high-performance banking dashboard optimizing user trust and accessibility (WCAG AA). We implemented bespoke framer-motion animations, Chart.js visualizations, and aggressive preloading to maintain a sub-80KB JavaScript payload for complex financial data.
