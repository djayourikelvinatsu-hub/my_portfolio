## Overview

Migrated a monolithic SPA to Next.js App Router, combining React Server Components and Edge rendering to drastically improve performance and SEO. We moved complex state (like carts) from Redux/LocalStorage to server-side HTTP cookies and Server Actions, achieving a 93% reduction in bundle size and an 81% faster LCP.
